# Snippet Group Command

よく使う定型文をグループ化して管理し、必要なときに呼び出してクリップボードにコピーするためのコマンド  

入力欄にグループ名を入力すると、そのグループに属する定型文の一覧が候補として表示される。  
候補を選択すると、その定型文がクリップボードにコピーされる。

![](../image/snipgroup.png)

## Description

![](../image/new-snippetgroupcommand.png)

- `グループの名前`  
定型文グループに属する定型文の一覧を表示するためのキーワード  
表示された候補を選択し、実行すると、該当する定型文の内容がクリップボードにコピーされる。
- `グループ説明`  
入力画面でのキーワード入力時にコメント表示欄に記載される文字列。
- `ホットキー`  
[コマンドを呼び出すキー](/window/command-hotkey)を設定できる。設定したキーを押すと、[入力画面](/usage/how_to_use.md#open-input-window)が表示され、グループの定型文の一覧が表示された状態になる
- `定型文の一覧`  
グループに属する定型文の一覧が表示される  
`F4`キー入力で次の項目、`Shift`+`F4`キー入力で前の項目に移動することができる。
- `追加`  
グループに定型文を追加する。  
`定型文の一覧`に新しい項目が追加される。
- `削除`  
グループにある既存の定型文の内容を削除する。
- `上へ移動`  
クリックすると、`定型文の一覧`で選択した項目を一つ上に移動する。
- `下へ移動`  
クリックすると、`定型文の一覧`で選択した項目を一つ下に移動する。
- `定型文の名前`  
定型文の名前を設定する。ここで指定した名前が候補欄に表示される。
- `定型文説明`  
該当する定型文が候補欄で選択されたときにコメント表示欄に表示する文字列
- `テキスト`  
定型文の内容を入力する

`定型文の名前`、`定型文説明`、`テキスト`のいずれかが選択されている(カーソルがある)とき、`Alt`+`↑`、または、`Alt`+`↓`キー入力により、これらの入力欄を移動できる。

## Per-Key Behavior

|押下キー|動作|
|--|--|
|`Enter`|設定された定型文をクリップボードにコピーする|
