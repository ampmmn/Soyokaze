# Switch PowerPoint Slide

入力画面を表示した時点でアクティブなPowerPointプレゼンーションのスライドタイトル一覧を候補に表示する。  
スライドタイトルで候補の絞り込みを行える。コマンド実行するとPowerPointのウインドウをアクティブにしたうえで、該当するスライドを表示する。  

![](../image/pptjump.gif)

初期設定で機能は無効化されており、[アプリケーション設定](/window/app-settings.html#extensions-switch-window)から有効化する必要がある。(`アプリ設定`>`拡張機能`>`ウインドウ切替`)
プレフィックスは初期設定で`pj`

`Ctrl-Enter`キーを押して実行すると、最大化表示する。
