# Switch PowerPoint Slide

入力画面を表示した時点でアクティブなPowerPointプレゼンーションのスライドタイトル一覧を候補として表示する機能。  
プレフィックス(初期設定で`pj`)を入力すると、表示中のスライドタイトル一覧が候補として表示され、スライドタイトルで候補の絞り込みを行える。コマンド実行するとPowerPointのウインドウをアクティブにしたうえで、該当するスライドを表示する。  

![](../image/pptjump.gif)

初期設定で機能は無効化されており、[アプリケーション設定](/window/app-settings)の`拡張機能`>`ウインドウ切替`から有効化する必要がある。

`Ctrl-Enter`キーを押して実行すると、最大化表示する。
