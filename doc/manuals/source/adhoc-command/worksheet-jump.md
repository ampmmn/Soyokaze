# Switch Excel/Libre Office Calc Worksheet

入力画面を表示した時点で開いているExcel(デスクトップ版)とLibreOffice Caclのシート名一覧を候補に表示する。  
シート名で候補の絞り込みができる。コマンドを実行すると、対応するワークシートのウインドウをアクティブにしたうえで、シートを切り替える。

![](../image/calc-sheet.gif)

初期設定で機能は有効化されており、機能を利用しない場合は[アプリケーション設定](/window/app-settings.html#extensions-switch-window)から無効化できる。(`アプリ設定`>`拡張機能`>`ウインドウ切替`)
プレフィックスは初期設定で`xj`

`Ctrl-Enter`キーを押して実行すると、最大化表示する。
